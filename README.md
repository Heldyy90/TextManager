## 📌 Описание

**TextManager** — это Fabric-мод для Minecraft **1.20.1**, созданный для модераторов и игроков на **HolyWorld**.

Мод ускоряет работу в чате: позволяет хранить готовые сообщения, команды и шаблоны, а затем отправлять их в один клик без ручного ввода.

Он особенно удобен для модераторов, которым часто нужно отправлять одинаковые сообщения, предупреждения, команды или ответы игрокам.

## ✨ Возможности

| Возможность | Описание |
|---|---|
| 🗂️ Заготовки сообщений | Хранение часто используемых сообщений, ответов и служебных команд |
| ⚡ Отправка в 1 клик | Нажатие по названию шаблона сразу отправляет текст в чат |
| 🎨 Форматирование | Поддержка цветов и стилей: `&c`, `&a`, `&l`, `&n`, `&o` и другие |
| 🔧 Управление шаблонами | Добавление, редактирование, переименование и удаление шаблонов |
| 📖 Встроенная справка | Команда `/textshelp` открывает окно со всеми командами и примерами |
| 🔄 Живое обновление конфига | Конфиг применяется без перезапуска игры |
| 🧩 Отдельный чат и фильтры | Сообщения TextManager можно вынести в отдельный чат и настроить фильтрацию |
| 🛠️ Удобство для модераторов | Быстрая отправка одинаковых сообщений экономит время при проверках и общении |

---

## ⌨️ Команды

| Команда | Назначение |
|---|---|
| `/textadd` | Добавить новый шаблон |
| `/textedit` | Изменить существующий шаблон |
| `/textrename` | Переименовать шаблон |
| `/textremove` | Удалить шаблон |
| `/textsconfig` | Загрузить или обновить конфиг |
| `/textshelp` | Открыть справку по командам мода |
